MyST Studio User Guide v5.8.6

What's new in v5.8.6

  • Bug fix: Fixed issue where the SSH private keys could be accessed through the file system and the support artifact, thus leading to a security vulnerability
  • Bug fix: Fixed a synchronization issue around Stream Model Properties
  • Improvement: Now MyST will provide an option to disable version conflict validation for a Release Pipeline
  • ODI and OEDQ 12.2.1.3.0 configurations can now be created in MyST Studio. Software installation and domain creation is supported.
  • Added 12.2.1.3.0 support for MyST Build Extensions
  • Bug Fix: Fixed an issue due to which failed stream revisions from previous stage were appearing as available for deployment in current stage. It has been ensured if a stream revision fails to deploy at 1st stage, it won't appear in the 2nd stage as a valid deployment candidate.
  • Fixed an regression originally caused by fix
  • WCC, WCC UI and IBR 12.2.1.3.0 configurations can now be created in MyST Studio. Software installation and domain creation is supported.
